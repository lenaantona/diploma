Приложение для сотрудника центра занятости населения
Приложение не подразумевает аутентификацию, тк будет использована одним сотрудником и локально.
Для учета активной политики занятости, а именно учета договоров, заключенных на определенные виды работ
сущности: договора(contract), работодатели(employer), работники(worker), виды работ(typework), contractworker - связывает договор и работника
в договоре есть ссылка на работодателя и виды работ.

server -  Java Spring Boot
client -  html + bootstrap
bd - MySQL
приложен файл - aep.sql (для создания БД)
начальная страница - http://localhost:8080/index
все контролеры имеют в названии HTML
